About Me

A dedicated researcher advancing hardware acceleration and neural network optimization

My research interests include designs of machine-learning hardware accelerators, compiler migration, and deep neural network compression. I focus on bridging the gap between software algorithms and efficient hardware implementation.

Research Interests

Exploring the frontiers of AI hardware and algorithm co-optimization

Quantization & Compression

Developing advanced quantization and compression methods for AI models, reducing model size and computational requirements while maintaining accuracy for efficient deployment on edge devices.

Hardware Acceleration

Designing efficient hardware accelerators for AI algorithms, including CNN inference accelerators with feature map compression and spiking neural network accelerators with spatio-temporal redundancy reduction.

Neural Network Optimization

Researching deep neural network compression techniques, including interlayer feature map compression based on least-squares fitting and intra/inter-channel feature map compression methods.

Compiler Migration

Working on compiler migration techniques to bridge the gap between high-level AI algorithms and efficient low-level hardware implementations, enabling seamless deployment across different hardware platforms.

Near-Memory Computing

Developing fast-convergence near-memory-computing accelerators for solving partial differential equations, pushing the boundaries of compute-in/near-memory architectures.

Spiking Neural Networks

Building energy-efficient SNN accelerators that exploit spatio-temporal redundancy for reduced computation, enabling ultra-low-power neuromorphic computing solutions.
